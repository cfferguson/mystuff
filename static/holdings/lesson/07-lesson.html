Kaggle

Kaggle1 is an online learning community for data scientists and machine learning practitioners2 That’s great but its greatest benefit are the numerous tutorials/walkthroughs provided and also to locate data sets3. In an upcoming data task, you’ll be asked to locate a data set of interest to perform some analysis and Kaggle is a great place to start so if you get a chance, sign up for free and look around. However for now, there is an excellent dplyr walkthrough that was posted recently and may be helpful to you for the exam.
